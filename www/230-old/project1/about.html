About this Project

Why this topic?

I chose this topic to help others learn how to play the guitar. One of the students housing in my dormitory this year is an avid player. He enjoys teaching others about music theory as well as showing others how to play the instruments he enjoys. He wants to start a special interest house on floor for music; he gave me the idea to do the tutorial on playing the guitar.

How did you go above and beyond?

I spent a lot of time tackling this from a mobile-first design; I listed my goals out and had responsiveness be the personal challenge I would tackle. Making use of CSS3 animations and JavaScript to fully utilize the website, I was able to create a sleek, appealing interface.

Anything special?

By using JavaScript, I was able to make a sticky sidebar navigation that affixes to the screen after scrolling past a certain point. I utilized the data- attributes supplied by HTML5 for use with the JavaScript scripts. I used a script to offset the content when the off-canvas navigation is used. The sidebar and off-canvas navigations show up on the desktop/large screen versions of the website.

How could you improve this site?

If I had the time, I would rework the way I did the CSS styling. I take an approach of experimenting and then refining, which can leave me with a good excess amount of classes that may not be used in the final product. I would like to be able to return and clean this up.

In addition to reworking the styling, I was also experimenting with javascript to trigger a change in the aside navigation panel where the "current" link would switch depending on how far down the page the user has scrolled. Unfortunately, my implementations were clunky, using data-set attributes that only worked for one screen size. Accounting for multiple sizes would have taken a bit more time, so, I removed the feature to prevent any confusion.

About the Class

IGME 230
Web Design & Implementation

In the School of Interactive Games and Media, IGME 230 is all about creating websites and crafting designs that follow established principles that we know. From CRAP to ERDU, tackling design is part of the assignments we face.

This course provides an introduction to web development tools and technologies, such as HTML, CSS, JavaScript and DHTML, AJAX, web platforms and environments, and server-side programming methods. Programming projects are required.
